# macvim256.vim

This is a port of the macvim theme modified for 256 color terminals.

    set background=light
    colorscheme macvim256

![MacVim Light](http://i.imgur.com/a0GUAl5.png)

    set background=dark
    colorscheme macvim256

![MacVim Dark](http://i.imgur.com/WH5C96w.png)
